Quick Guide to MecQBank

This page gives you a quick overview of how to use MecQBank to manage your subject's assessment problems and assemble assignments from them. MecQBank will provide you a searchable, traceable way to see what problems were used when, as well as what learning objectives they assess.

Subject Selection

On the left-hand side of this page, you see the subjects available to you, under the heading My Subjects.

Clicking on one of those subjects allows you access to its items, to be explained below in Item and Assignment Actions.

If you want access to another subject (Other Subjects), you need to request permission from one of its current owners. Clicking the subject brings up a list of users you can ask.

Item and Assignment Actions

Once you are working with a specific subject, the top of the page will show menu items for managing items and assessments within the subject. By default the Upload Items option is selected.

Uploading Items

You can use the browser's file selector or drag & drop the files into the target area. Item files are expected to match the LaTeX and PDF formats outlined in this Google document. Problem LaTeX and PDF files should be named consistently...for example:

  • CONT0001.tex
  • CONT0001.pdf
  • CONT0001_Sol.tex
  • CONT0001_Sol.pdf
  • CONT0001.meta
Along with any referenced image files.

The app automatically groups the files according to item name and does some basic validation on the uploaded data. If all the files and metadata seem to be present, you will have the option to Submit Item.

If the item was created correctly, you receive a verification message.

Otherwise, you see an error message with a brief description at the top of the page.
